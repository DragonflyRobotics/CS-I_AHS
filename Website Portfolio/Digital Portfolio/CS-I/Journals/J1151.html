1.2 Why do you think the term consequent is used to refer to the statements that are executed when the Boolean test evaluates to true? --> The term consequent is used because that is what the code will run as a consequence of the conditional being true.

2.1 Why is the first Jump instruction conditional? What determines whether the jump will be executed or not? --> It is testing to see if the input number is less than or equal to 4. Only then will it jump to the alternative function.

2.2 What is the purpose of the Jump instruction after the consequent? --> It is meant to jump to a function that will exit once the process the consequent had to do is complete.

2.3 Why is there no need for a corresponding Jump instruction after the alternative? --> Because the exit instructions come right after the alternative so there is no need to jump since it will already be in order.
