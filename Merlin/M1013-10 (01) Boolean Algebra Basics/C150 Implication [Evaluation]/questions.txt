1.  ¬true ← false 
2.  ¬true → ¬false 
3.  false → false 
4.  true ← false 
5.  false ← true 
6. ( false ⩒ true ∨ ¬false ) ↔ ( false ⊼ ¬false ⊽ ¬true )
7. ( ¬true ⊽ ¬false ⩒ true ) ↔ ( ¬true ∨ true ⊽ true )
8. ( ¬true ⩒ ¬true ∨ ¬false ) ↔ ( true ⊽ ¬true ⊽ false )
9. ( ¬false ⩒ ¬false ⊼ true ) ← ( true ⊽ true ⊼ ¬true )
10. ( ¬true ∨ true ∨ ¬false ) → ( false ⊽ true ⊼ ¬false )
