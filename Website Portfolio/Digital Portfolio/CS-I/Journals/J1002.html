7.1 How does the ls command differ from the tree command? --> The "tree" command will list all the files in the current directories and subdirectories in an organized, hiearchial manner. "ls", on the other hand, simply lists files in the current directory.

7.2 How do you choose to use one over the other? --> If you need to quickly navigate to a location that you are familiar with, you would use "ls". But, if you want to find a particular file or navigate unknown folders, you would use tree. It is also sometimes beneficial to not use tree since if a folder has hundreds of subfolders and files, it will bloat the entire terminal.
